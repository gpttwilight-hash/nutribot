# NutriBot — Питание & Тренировки

NutriBot — это Telegram Mini App для подсчета калорий, трекинга питания и отслеживания тренировок. Оснащен AI для анализа еды по фото и элементами геймификации (строка дней, очки опыта, уровни, достижения).

## Структура проекта

- `/backend`: FastAPI сервер с PostgreSQL, SQLAlchemy, Aiogram (для Telegram бота) и Celery (для push-уведомлений).
- `/frontend`: React + TypeScript + Vite + Tailwind CSS приложение, интегрирующее Телеграм WebApp SDK.

## Локальный запуск

1. **База данных и Redis**
   Убедитесь, что у вас установлен Docker. Запустите необходимые сервисы (включая БД) с помощью:
   ```bash
   docker-compose up -d db redis
   ```

2. **Запуск Backend'а**
   Перейдите в папку `backend`, создайте виртуальное окружение, установите зависимости и запустите:
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env # заполните .env ключами
   alembic upgrade head # Примените миграции
   uvicorn app.main:app --reload
   ```

3. **Запуск Frontend'а**
   Перейдите в папку `frontend`, установите пакеты и запустите:
   ```bash
   cd frontend
   npm install
   cp .env.example .env
   npm run dev
   ```
   
> **Совет:** Для быстрого запуска в режиме разработки (если БД и Redis уже запущены на портах по умолчанию) можно воспользоваться скриптом `./start_dev.sh` в корне проекта.

## Функционал

- **AI-трекинг**: Отправьте фото еды, чтобы определить КБЖУ (через OpenAI Vision API).
- **Поиск**: Локальная база популярных продуктов + OpenFoodFacts API.
- **Тренировки**: Календарь тренировок с автоматическим начислением XP.
- **Геймификация**: Ежедневный бонус, система достижений, стрик (серия подряд).
- **Подписка**: Интеграция с Telegram Stars (звёзды).
